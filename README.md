# Game-Snake
Game Snake Sederhana adalah proyek berbasis Java yang menampilkan permainan ular klasik. Pemain mengendalikan ular untuk memakan makanan, membuat ular tumbuh lebih panjang, dan menghindari tabrakan dengan dinding atau tubuhnya sendiri. Proyek ini dibuat sebagai latihan pemrograman dasar dan penerapan konsep OOP dalam Java.
